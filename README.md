# FRR_utils
Scripts and tools developed at the USDA Forage and Range Research Lab for bioinformatic analyses

## Introduction
This repository is a collection of utlities, each in a separate folder, that are used at the USDA ARS Forage and Range Research Unit (FRR) in Logan, Utah. Each utility is explained in its own directory.

## List of utilities
| Utility               | Description                                                   | Type          |
|-----------------------|---------------------------------------------------------------|---------------|
| add_entap_results2gff | Adding funtional annotation from [EnTAP](https://entap.readthedocs.io/en/latest/index.html) results to a .gff file | python script |
